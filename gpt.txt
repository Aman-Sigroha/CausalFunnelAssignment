# What You'll Build

Three separate things:

### 1. Tracking Script

Acts like Google Analytics.

Embedded inside any webpage.

Tracks:

```txt
page_view
click
```

and sends them to backend.

---

### 2. Analytics Backend

Stores everything in MongoDB.

Provides APIs for:

```txt
Save Event

List Sessions

Get Session Journey

Get Heatmap Data
```

---

### 3. Analytics Dashboard

Admin dashboard.

Displays:

```txt
Sessions
User Journey
Heatmap
```

---

# Database Design

Single collection.

### Event Model

```js
{
  _id

  sessionId

  eventType

  pageUrl

  timestamp

  clickX

  clickY

  userAgent

  viewportWidth

  viewportHeight
}
```

Store viewport size too.

Looks more professional.

---

# Feature 1 вҖ” Session Tracking

When page loads:

```txt
session_id = uuid
```

Store in:

```js
localStorage
```

Example:

```txt
7b4f82f1-a5e7-42f2-b3d1
```

Reuse for all future events.

---

# Feature 2 вҖ” Page View Tracking

When page opens:

```js
trackPageView()
```

Payload:

```json
{
  "sessionId": "...",
  "eventType": "page_view",
  "pageUrl": "/home",
  "timestamp": "..."
}
```

Send:

```http
POST /api/events
```

---

# Feature 3 вҖ” Click Tracking

Track every click.

```js
document.addEventListener("click")
```

Collect:

```js
event.clientX
event.clientY
```

Payload:

```json
{
  "sessionId":"...",
  "eventType":"click",
  "pageUrl":"/home",
  "clickX":420,
  "clickY":180
}
```

# Backend APIs

---

## API 1

### Create Event

```http
POST /api/events
```

Stores event.

---

## API 2

### Sessions Summary

```http
GET /api/sessions
```

Response:

```json
[
  {
    "sessionId":"123",
    "eventCount":15,
    "firstVisit":"...",
    "lastVisit":"..."
  }
]
```

Use MongoDB aggregation.

---

## API 3

### Session Journey

```http
GET /api/sessions/:id
```

Returns:

```json
[
 page_view
 click
 click
 page_view
]
```

ordered by timestamp.

---

## API 4

### Heatmap Data

```http
GET /api/heatmap?page=/home
```

Returns only clicks.

---

## API 5 (Bonus)

### Unique Pages

```http
GET /api/pages
```

Response:

```json
[
 "/home",
 "/products",
 "/pricing"
]
```

Useful for heatmap dropdown.
